# Ember

Ember audio player. Capable of playing mp3, wav, ogg, or flac files from your pc  
Mainly made for personal use

## Installation
```
git clone https://github.com/Caleb-Kronstad/Ember
```

## Use
Run ```Setup.bat```, Build via IDE or Build system, run Ember.exe  
Alternately, download Ember.zip from the [releases page](https://github.com/Caleb-Kronstad/Ember/releases) of the Ember GitHub, then run Ember.exe

Made using [Kiln](https://github.com/Caleb-Kronstad/Kiln)
- Miniaudio
- Dear ImGui
- Glad
- GLFW
- Nlohmann Json
